1.查询所有商品的平均价格 select avg(sprice) as averagep from shangpin; 2.查询某一类型商品的平均价格 select avg(sprice) as averagep from shangpin where stypeid=1; 3.查询某个商品的存货价值 select sname,sprice*snumber as value from shangpin; 4. 查询某个订单的金额 select shangpin.sprice*sorder.number from sorder,shangpin where shangpin.id=1; 5.查询价格最低的三个商品 select * from shangpin order by sage limit 0,3; 6.查询最新添加的一个商品 select * from shangpin order by id desc limit 0,1; 7.查询商品的所有信息以及它的类型 select shangpin.*,type.tname from shangpin,type where shangpin.stypeid=type.id; 8.查询销量最高的三个商品 select shangpin.sname,sorder.number from shangpin,sorder where shangpin.id=sorder.sid order by sorder.number desc limit 0,3;
